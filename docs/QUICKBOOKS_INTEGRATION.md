# QuickBooks Integration

Ceiba Data AI Explorer uses Stripe for billing. QuickBooks syncs automatically via:

## Option A — Stripe App for QuickBooks (Recommended, Free)
1. Go to: apps.quickbooks.com → search "Stripe"
2. Install the official Stripe app
3. Connect your Stripe account
4. All Stripe invoices, payments, and customers sync automatically to QuickBooks

## Option B — Synder (Third-party, $65/month)
synder.com — more advanced sync with reconciliation

## What Syncs Automatically
- Customer records (hospital name, email)
- Invoices (annual and monthly)
- Payments received
- Failed payments / refunds
- Subscription changes

## Stripe Dashboard
https://dashboard.stripe.com — view all subscriptions, invoices, and customers
